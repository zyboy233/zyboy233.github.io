部署脚本&启动服务：

 > weget https://github.com/Wind4/vlmcsd/releases/tag/svn1112
 > tar -xzvf binaries.tar.gzip
 > ./binaries/Linux/intel/static/vlmcsdmulti-x64-musl-static

客户端测试：

> vlmcs.exe -v -l 3 zyboy.top:1688

-v 输出详细信息

-l 3 发送Windows Server 2008 Datacenter的激活请求，具体版本对应列表可以通过vlmcs.exe -x查看www.zyboy.top 表示KMS服务器的域名

Win10 vol 版激活

W269N-WFGWX-YVC9B-4J6C9-T83GX – 安装密钥,此密钥用于win10专业版使用

slmgr /upk   卸载密钥

cd /d "%SystemRoot%\system32" 

slmgr /skms 你的KMS服务端主机的IP或者域名

slmgr /ato 

slmgr /xpr

Office激活

清除激活信息

cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /dstatus

如果显示有kms等字样，说明是KMS激活的，请继续以下步骤：

cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /unpkey:AB12C

(上面的KEY的最后5位)

cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /remhst

Office vol 激活

cd "C:\Program Files\Microsoft Office\Office16" 

cscript ospp.vbs /sethst:zyboy.top:1688

cscript ospp.vbs /act

重启


linux win
