Redis 是完全开源免费的，遵守BSD协议，是一个高性能的key-value数据库。

Redis 与其他 key - value 缓存产品有以下三个特点：

  • Redis支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。
  • Redis不仅仅支持简单的key-value类型的数据，同时还提供list，set，zset，hash等数据结构的存储。
  • Redis支持数据的备份，即master-slave模式的数据备份。

redis数据库下载安装

  1. 下载地址: redis
  2. 解压安装包
  3. 将解压出来的文件夹改名为redis
  4. 修改或添加redis数据库的启动配置文件redis.windows.conf如下内容:
    \> # bind 127.0.0.1  ------关闭绑定本地环回地址
\> protected-mode no ------关闭保护模块
\> daemonize yes     ------打开守护进程
\> Ctrl + s ------保存
  5. cd到redis目录执行执行以下命令开启本地服务器监听(注意:启用本地配置文件启动redis服务端):
    \> redis-server redis.windows.conf
  6. 打开另一个cmd终端cd到redis目录执行以下命令连接redis数据库并写入数据和取数据
    \> redis-cli -h 127.0.0.1
\> set task1 spider_baidu
\> set task2 spider_taobao
\> get task1`
\> get task2`
  7. 配置环境变量
  8. 局域网下连接server服务器,需指定本地局域网下server服务器ip地址
    \> redis-cli -h 服务器ip
  9. 这时就能访问redis服务器读写数据了
